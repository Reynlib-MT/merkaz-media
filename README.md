# Merkaz Media — сайт

Одностраничный лендинг (иврит, RTL).

## Структура

```
merkaz-media/
├── index.html      ← главная страница
├── styles.css
├── script.js
├── assets/         ← картинки и логотипы
└── archive/        ← старые версии и материалы
```

## Работа с GitHub

1. Редактируете файлы в корне (`index.html`, `styles.css`, …)
2. VS Code → Source Control → Commit → Push
3. GitHub обновится автоматически

## GitHub Pages

Settings → Pages → Branch: `main`, folder: `/ (root)`  
Сайт будет на: `https://reynlib-mt.github.io/merkaz-media/`
